# Changelog

## Version 1.0

* Add functions to read from and write to a stream. Search the documentation for
  `context-read-from-stream!`, `context-write-to-stream!` and also for
  `with-stream-reader`.

## Version 0.1

* Reading and writing toplevel images, currently only 8 bits per color (so 8, 24
  or 32 bits per channel).
* Reading metadata
* Encoders queries
